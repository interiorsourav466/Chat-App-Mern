# MERN Realtime Chat App

A full-stack realtime chat application built using MERN stack and Socket.io.

## Features

- User authentication (JWT)
- Realtime messaging using Socket.io
- Profile image upload using Cloudinary
- Online/offline user status
- Modern UI using Tailwind CSS
- Image sharing in chat
- Responsive design

## Tech Stack

Frontend:
- React.js
- Tailwind CSS
- Zustand
- Axios

Backend:
- Node.js
- Express.js
- MongoDB
- Socket.io
- Cloudinary

## Installation

### Backend

cd backend
npm install
npm run server

### Frontend

cd frontend
npm install
npm run dev
